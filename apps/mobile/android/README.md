# Android native project placeholder

This directory is reserved for the Expo-generated Android native project.

The selectable message text bridge is generated by
`plugins/withSelectableMessageTextAndroid.js`. The generated Kotlin files under
`android/app/src/main/java/...` are local build output and should not be
committed directly.

From `apps/mobile`, regenerate the Android project when needed:

```sh
npx expo prebuild --platform android
```

For normal local Android work, `npm run android` can use or regenerate the
native project as needed.
